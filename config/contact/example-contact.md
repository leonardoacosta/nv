# Contact Profile Template

> Files prefixed with `example-` are excluded from Nova's system context injection.
> Copy this file to `<slug>.md` (e.g. `leo-acosta.md`) and fill in the details.
> The filename slug should match the contact's name in kebab-case.

## Identity

**Name:** Full display name (matches the `name` field in the contacts DB row)

**Channels:**
- Telegram: `@handle` or first name
- Discord: user ID (numeric string)
- Teams: UPN (`user@domain.com`)

## Relationship

**Type:** `work` | `personal-client` | `contributor` | `social`

**Context:** One or two sentences about how this person relates to the operator.

## Communication Style

- Preferred style (casual / formal / technical)
- Response time expectations
- Preferred message length

## Timezone & Availability

- **Timezone:** e.g. `America/Chicago (CST, UTC-6)`
- **Work hours:** e.g. `Mon–Fri 09:00–18:00 CST`
- **Do not disturb:** e.g. `after 21:00 on weekdays`

## Ongoing Projects

List any projects, topics, or epics this contact is involved with.
Nova will use this to provide relevant context in replies.

## Notes

Any additional operator notes: travel, deadlines, sensitive topics to avoid, etc.
